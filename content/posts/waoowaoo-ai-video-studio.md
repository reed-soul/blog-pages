## 一、背景：AI 视频制作的痛点

随着短视频平台的兴起，越来越多的人想要制作自己的视频内容。但传统的视频制作流程复杂且耗时：

1. **剧本创作**：需要专业的编剧技能
2. **角色设计**：需要美术功底
3. **场景绘制**：需要背景设计能力
4. **分镜制作**：需要导演思维
5. **配音录制**：需要配音演员或专业设备
6. **视频剪辑**：需要剪辑软件和时间

对于小说作者、漫画家、内容创作者来说，他们有好的故事，但缺乏视频制作的技术和资源。

这时候，AI 技术可以帮上大忙。

## 二、waoowaoo 是什么？

**waoowaoo** 是一款开源的 AI 影视 Studio，它的核心能力是：**从小说文本自动生成分镜、角色、场景，并制作成完整视频**。

简单来说，你只需要提供一段小说文本，waoowaoo 就能帮你：

- 🎬 **AI 剧本分析**：自动解析小说，提取角色、场景、剧情
- 🎨 **角色 & 场景生成**：AI 生成一致性的人物和场景图片
- 📽️ **分镜视频制作**：自动生成分镜头并合成视频
- 🎙️ **AI 配音**：多角色语音合成

更棒的是，它是 **完全开源的**，你可以在自己的电脑上运行，无需担心数据隐私。

## 三、快速开始：5 分钟体验

### 前提条件

只需要安装 [Docker Desktop](https://docs.docker.com/get-docker/)。

### 一键启动

```bash
# 克隆项目
git clone https://github.com/waoowaooAI/waoowaoo.git
cd waoowaoo

# 启动服务
docker compose up -d
```

访问 [http://localhost:13000](http://localhost:13000) 即可开始使用！

首次启动会自动完成数据库初始化，无需任何额外配置。

### 配置 AI 服务

进入**设置中心**，配置 AI 服务的 API Key：

- **推荐**：字节火山引擎（Seedance, Seedream）
- **备选**：Google AI Studio（Banana）
- **文本模型**：需要 OpenRouter API

项目内置了配置教程，跟着步骤操作即可。

## 四、技术架构：如何实现 AI 视频生成

waoowaoo 的技术架构非常值得学习，它展示了如何构建一个现代化的 AI 应用。

### 核心技术栈

| 技术选型 | 用途 |
|---------|------|
| **Next.js 15** | 前端框架 |
| **React 19** | UI 库 |
| **MySQL** | 数据存储 |
| **Prisma ORM** | 数据库操作 |
| **Redis** | 缓存和队列 |
| **BullMQ** | 任务队列 |
| **Tailwind CSS v4** | 样式系统 |
| **NextAuth.js** | 用户认证 |

### 架构设计

```
┌─────────────────┐
│   Next.js App   │  ← 前端界面
│   (React 19)    │
└────────┬────────┘
         │
         ├─→ MySQL (数据存储)
         │   └─ 用户、项目、素材
         │
         ├─→ Redis (缓存 & 队列)
         │   └─ BullMQ 任务队列
         │
         └─→ AI Services (外部服务)
             ├─ 文本分析（LLM）
             ├─ 图片生成（SD/DALL-E）
             ├─ 语音合成（TTS）
             └─ 视频合成（FFmpeg）
```

### 核心模块

1. **AI 运行时抽象层**（`src/lib/ai-runtime`）
   - 统一不同 AI 服务商的接口
   - 支持热切换服务商

2. **任务队列系统**（`src/lib/workers`）
   - 视频生成是耗时操作
   - 使用 BullMQ 异步处理
   - 支持任务重试和错误处理

3. **多媒体处理**（`src/lib/media`）
   - 图片生成和管理
   - 语音合成
   - 视频合成

4. **国际化支持**（`src/i18n`）
   - 中文/英文界面
   - 一键切换

## 五、核心功能详解

### 1. AI 剧本分析

waoowaoo 使用大语言模型（LLM）分析小说文本，提取：

- **角色信息**：姓名、外貌、性格、服装
- **场景描述**：地点、时间、氛围
- **剧情结构**：起承转合、关键情节

这个过程完全自动化，你只需要粘贴文本。

### 2. 角色 & 场景生成

基于提取的信息，waoowaoo 会调用 AI 绘图模型：

- **角色一致性**：同一个角色在不同场景中保持外貌一致
- **场景连贯性**：场景风格统一
- **自动适配**：根据剧情需要生成不同表情和动作

### 3. 分镜视频制作

waoowaoo 会自动：

1. 将剧情拆分成多个分镜
2. 为每个分镜生成画面
3. 添加过渡效果
4. 合成完整视频

### 4. AI 配音

支持多角色语音合成：

- 不同角色使用不同的声音
- 支持调节语速和情感
- 自动匹配角色性格
